## Description 

 With wp-optimizer you will be able to optimize your WordPress database and your images.
 The Image Optimizer will increase your page speeds by way of image optimization driving to better search engine rankings and save you storage space and bandwidth. 
 With Image optimizer you will be able to will optimize any images uploaded and created by any plugin already existing and automatically optimize any new image daily.

### Why use Database Optimizer?

1. **Space** After days of use WordPress will populate the database with some un useless data.
1. **Speed** Optimize the database will drive to a faster loading WordPress

### Why use Image Optimizer?

1. **Privacy** Any image will be optimized on your server using (jpegtran, optipng, pngout, pngquant, gifsicle, cwebp)
1. **No Speed Limits** and unlimited file size.
1. **Smooth Handling** with pixel-perfect optimization using industry-leading tools and progressive rendering.
1. **Free** The plugin is free and always will be.

Speed up your website to better connect with your visitors. Properly compress images. Includes lazy load, optimize the database and so on.

## Installation 
Upload this plugin in the directory '/wp-content/plugins/' or download this from your WordPress admin panel.
Activate this plugin from the 'Plugin' menu inside WordPress.
